Willkommen zum Experiment!

Für die Teilnahme an dieser Studie brauchen Sie einen Rechner mit Tastatur. Ihre Aufgabe in diesem Experiment ist es, Sätze zu lesen. Diese werden Ihnen Stück für Stück angezeigt. Zuerst erscheint in der Mitte des Monitors ein Asterisk (*). Sobald Sie die LEERTASTE drücken, sehen Sie einen Satz. Sie sehen zuerst nur Striche, die anzeigen, wie lang der Satz jeweils sein wird:

____ _____ _____ ___ ___ ______

Um den ersten Teil des Satzes zu sehen, drücken Sie bitte die LEERTASTE:

Eine Katze _____ ___ ___ ______

Sobald Sie wieder die LEERTASTE drücken, erscheint der nächste Teil und der vorhergehende verschwindet wieder, usw.

____ _____ sitzt ___ ___ ______
____ _____ _____ auf ___ ______
____ _____ _____ ___ der Decke.

Lesen Sie die Sätze in Ihrem eigenen, normalen Lesetempo, sodass Sie den Inhalt verstehen. Bitte lassen Sie während des Lesens immer einen Finger auf der LEERTASTE. Betätigen Sie stets die LEERTASTE, nachdem Sie einen Satzabschnitt gelesen haben, um den nächsten zu sehen.

Auf manche Sätze folgt eine Frage mit zwei Antwortalternativen, die sich ausschließlich auf den zuvor gelesenen Satz bezieht. Bitte beantworten Sie diese Frage indem Sie entweder auf die entsprechende Antwort klicken oder die Zahl 1 (=LINKS) oder 2 (=RECHTS) drücken. Eine einmal gegebene Antwort können Sie nicht mehr revidieren. Machen Sie einfach mit dem nächsten Satz weiter. Die Fragen sehen immer so aus:

Macht dieser Satz Sinn?
1 = nein              2 = ja

Zuerst werden Sie die Aufgabe anhand von 9 Beispielen üben. Dann folgt das eigentliche Experiment. Wenn Sie eine Pause brauchen, können Sie diese machen, während der Asterisk (*) erscheint. Am Ende werden Sie Ihren persönlichen Code für die Vergütung bekommen sowie eine kurze Erklärung zum Zweck dieser Studie lesen. Das gesamte Experiment dauert etwa 45 Minuten. Die genaue Dauer hängt davon ab, wie schnell Sie lesen und ob Sie Pausen einlegen.
